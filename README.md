## Development

### Prerequisites

- nodejs 14.15.1
- hardhat 2.3.0

### Install Dependencies

```bash
npm install
```

### Compile

```bash
npx hardhat compile
```

### Running the tests

```bash
npx hardhat test
```

### Deploy

```bash
npx hardhat run scripts/deploy.js --network rinkeby/ropsten/...
```
